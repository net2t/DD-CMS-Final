# GITHUB RULES (For Non-Coders)

## Purpose

Yeh file un logon ke liye hai jo coding experience nahi rakhte, taki woh asaan urdu/english mein samajh saken ke Github par code kaise push karna hai aur kaunse rules follow karne hain.

---

## Basic Rules

- **Har Modification ya Addition ke baad:**
  - Hamesha code ko Github par commit karo.
  - Commit message AI ki taraf se dena zaroori hai (har commit mein).
  - Agar chahein toh temporary branch bana kar bhi commit kar saktay hain.
  - Local files bohat important hain, unko protect karo.
  - Agar Github ki taraf se koi changes aayein (pull/merge conflicts), toh bina confirmation ke delete na karo. Pehlay mujh se pooch lo.
  - Har bug/error/modification/addition ke baad, usay apni taraf se script Run karo. Test karo Verify the issue resolved. Assum Pending waiting for approval. jab tak main allow/approve na karoon.
  - Test run hamesha 3-4 profiles pe karo, unlimited scraping na chalao.

---

## Commit/Push Instructions (AI/Developer Terms)

1. **Code Change ya Naya Feature:**
    - `git add .`
    - `git commit -m "<AI generated message>"`
    - `git push origin <branch-name>`
    - (Zaroori: Commit message AI generate kare)
2. **Temporary Branch:**
    - `git checkout -b <temporary-branch>`
    - Code changes karo, phir commit/push karo.
3. **Github Changes/Conflicts:**
    - Agar koi file ya code Github ki taraf se aata hai (pull/merge), delete/overwrite na karo. Pehlay mujh se confirm karo.
4. **Testing:**
    - Scraping ya code test karte waqt sirf 3-4 profiles pe test karo, unlimited profiles pe nahi.
5. **Suggestions / Improvements:**
    - Dont Try to Improve by your end. Always Give details about it and benifits of it. On approval you can implement it.
6. **System Message:**
    - Act like a Teacher and also help me to learn the code from very basic level.

---

## General Workflow

- Koi bhi bug/error/issue bataya jaye, bina poochay turant fix karo. (AI/Developer ko "would you like me to implement" ya "main aisa karoon?" poochne ki zarurat nahi.). No need to ask for permission.
- If there will be any problem then I will reverse it.
